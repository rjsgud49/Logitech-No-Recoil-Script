🔥 v1사용법

반동 강도 증가: 엄지버튼 4로 반동 강도를 증가시킬 수 있음.

반동 강도 감소: 엄지버튼 5로 반동 강도를 감소시킬 수 있음.

매크로 토글: 마우스 휠클릭으로 반동제어 기능을 켜고 끔.

좌클릭 시 반동제어 적용: MoveMouseRelative(0, recoilStrength)로 반동 강도에 따라 마우스를 이동.

매크로 토글: 마우스 휠클릭으로  반동제어 기능을 켜고 끔.

좌클릭 시 반동제어 적용: MoveMouseRelative(0, recoilStrength)로 반동 강도에 따라 마우스를 이동.
